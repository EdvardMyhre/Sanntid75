Check branches:
git branch
git branch -a		(shows external)

Change branch:
git checkout BRANCH_NAME

Check changelog:
git status

Save changes locally:
git commit -a -m "COMMIT MESSAGE HERE"


Get branch from server:
git pull origin BRANCH_NAME

Push to server:
git push origin BRANCH_NAME
